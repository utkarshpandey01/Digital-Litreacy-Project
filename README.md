# Digital-Litreacy-Project

Introduction

This repository contains my Digital Literacy Project submitted as part of the CSE0001 course. The main objective of this project was to understand the importance of digital literacy and learn how to use various digital platforms and tools effectively and responsibly. Through this project, I learned about creating digital content, building a professional online presence, online communication, collaboration tools, and cyber safety.

# Task 1 – Infographic
In this task, I created a Digital Literacy Awareness infographic using Canva. The infographic includes topics such as safe internet practices, digital tools for students, and email etiquette. This task helped me understand how to present information visually in a simple and attractive way.

# Task 2 – Digital Portfolio
In this task, I created my digital portfolio by setting up my profiles on GitHub, LinkedIn, and Kaggle. These platforms will help me showcase my projects, connect with professionals, and improve my technical skills over the next four years.

# Task 3 – Coding & Collaboration Platforms
In this task, I created an account on HackerRank and solved a beginner-level programming problem. I also created a Google Form titled “Digital Literacy Awareness Quiz” and collected responses. The responses were automatically stored in Google Sheets, which helped in organizing and analyzing the data.
